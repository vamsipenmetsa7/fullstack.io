Why Vue Is Awesome


Vue is a web framework that is as if you rewrote AngularJS 1, kept the good parts, and integrated what we learned from React. I think of it as sort of a combination between AngularJS 1 and React.


Vue is also the fastest growing JavaScript web framework -- and it’s worth learning, even if you can’t rewrite your whole app right now.


​Beyond Vue’s core, there are tools and libraries that exist in the Vue ecosystem that’s often needed to build real-world production scale applications. Things like client-side routing between pages, managing complex state, and form handling.

Frequently Asked Questions


Is this course right for me?



This class is designed for someone who wants to learn the ins-and-outs of building applications with the Vue.js framework.
- This class is hands-on. Roughly 60% lecture and 40% workshop.
- You do need to know some JavaScript to be able to complete this course
- You don’t need to be a JavaScript expert, as we will explain everything we cover in detail.
- You don’t need to be a Vue.js expert.
- We don’t assume any prior Vue.js knowledge. If you’ve used Vue a little bit, we build up to some slightly more advanced concepts (e.g. state management best principles) that might be useful to you.

If you want to get up an running with Vue quickly and not have to track down 30 different blog posts to figure it out, this class is for you.
